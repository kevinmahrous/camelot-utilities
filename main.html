About

This site is designed to make your trading commands a lot easier. Each specific tool is called a "Utility". While you might find similar utilities elsewhere, you'll often require opening 2 or 3 tabs (2 or 3 sites).

I've gathered everything in one place to simplify things for you. Plus, some tools, like the Separating Utility, are custom-made for Camelot. Check out the next few cards for a quick overview of what each utility does.

Joining Utility

The Joining Utility converts newline list (like a list you copy from the /inventory command) into a comma-separated list. For example:
Isolde EX
Artemis EX
Apollo EX
Becomes:
Isolde EX, Artemis EX, Apollo EX
This format can be used either for posting as an ad in a trading channel or with the /give characters command.

Separating Utility

The Separating Utility is the coolest. You can provide either a newline list or a comma-separated list, and it will separate them and return commands that you can use directly in the /give characters command. For example, inserting:

Counting Utility

The Counting Utility counts the number of characters in either a newline list or a comma-separated list. For example:
Isolde EX, Artemis EX, Apollo
Would count as 3.
It also counts duplicates, such as:
Isolde EX | 4x, Artemis EX, Apollo EX
Would count as 6.

Privacy

The site prioritizes your privacy by ensuring that all data remains on your device. Fonts, images, and other assets are stored locally, so no information is sent to external services like Google Fonts. I use Vercel (free plan) for hosting, and only non-identifiable data is collected for analytics purposes.
